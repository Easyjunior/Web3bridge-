A list of 30 HTML tags and their respective functions

  1. <!DOCTYPE>The Doctype declaration defines that this document is an HTML5 document
  2. <html>It is the root of HTML document
  3. <head>It defines the head of an HTML document that contains non-visible data like metadata and other information
  4. <body>It defines the body of a webpage and contains everything that you see on the webpage
  5. <p>It defines the paragraph in a webpage
  6. <br>It defines a line break
  7. <hr>It creates a horizontal line in a webpage
  8. <b>It is used to define bold text
  9. <i>It formats text in italic form
  10. <u>It underlines the text
  11. <sup>It defines superscripted text
  12. <sub>It defines subscripted text
  13. <pre>It defines preformatted text
  14. <em>It represents stress emphasis on the text same as i tag
  15. <del>It is used to display the text that has been removed from the webpage
  16. <ins>It defines a word that has been inserted in the document. It underlines the word
  17. <mark> It is used to highlight specific text
  18. <img>It defines image on webpage
  19. <figure>It defines a self-contained container for image, code snippets, diagrams, etc
  20. <thead>It groups the head content of the table
  21. <tbody>It groups the body content of the table
  22. <tfooter>It groups the footer content of the table
  23. <tr>It defines a row in the table
  24. <th>It defines a header cell in the table
  25. <td>It defines a cell in the table
  26. <div>It defines a group of elements within the HTML document
  27. <kbd> is used to display keyboard input on the screen. Browser display it by adding some default style to it.
  28. <blockquote>itindicates that the enclosed text is extended text from another source.
  29. <address>is used to display the contact or address of a person or an organization.
  30. <cite>describes a reference to a cited creative work or art. It must include the title of that work.
